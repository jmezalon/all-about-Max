NotesApp

NotesApp is a basic note taking app that allows users to add notes, to their notebooks. They can perform CRUD action for both notes and notebooks.

Group project built with a team of two within one week to showcase understanding of a basic full-stack app using sinatra and active record for the backend and react for the frontend.

Worked extensively in most of the backend, and did some fetch request in the frontend such as creating, deleting and updating the notes and notebook.

Contact Book Mockup

A contact book where users can save names, phone number, image, email and favorite. They are able to filter by favorite or all contact; and they can also use the search to find contacts.

A 3-days long project, built independently.

Created a json database file and made fetch request to get all contact, delete contact and finally add or remove contact from favorite.
